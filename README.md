# League of Legends Champion Viewer

A React application made for a university finals assignment. View champions from the popular online game [League of Legends](https://www.leagueoflegends.com) and save your favorites to view them later. Retrieves data from the [official API](https://developer.riotgames.com/docs/lol). You can view the live webpage on [github pages](https://lukatsanavabtu.github.io/react-finals-app/).

<br/>

Some of the features are: 
- Browsing champions;
- Searching champions;
- Adding/removing champions from favorites list;
- Viewing detailed champion info in favorites list;
- Persistent favorites list storage;
- Responsive design and transitions.

<br/>

Technologies used:
- React Hooks;
- React Router;
- Typescript;
- Vercel SWR;
- Tailwind CSS;
- Transitions;
- LocalStorage;
- Dialogs.
